Тест хедера Telegram Web K

Этот хедер имеет следующие особенности:

  • Высота 46 пикселей
  • Логотип Telegram с шрифтом Eurocine
  • Надпись "*неофициальная версия" поверх текста Telegram
  • Кнопка "Купить 50 звёзд за 79 руб." со звездочкой по центру
  • Поле поиска "База знаний..." с символом "/" справа
  • Кнопка переключения темы справа
  • Адаптивный дизайн для мобильных устройств
  • Цвета: #3390EC для светлой темы, #2B2D2F для темной

Функциональность

Нажмите на кнопку переключения темы в правом верхнем углу, чтобы переключиться между светлой и темной темами.

Адаптивность

Попробуйте изменить размер окна браузера, чтобы увидеть, как хедер адаптируется к разным размерам экрана.
